Node 地下铁沙龙 #10

Let’s imagine 2021!

Erwan Hesry

这是一次面向2021的地下铁沙龙，

我们精选了5个社区有代表性的开源项目，

深入 Node.js 相关领域，来讨论他们眼中的过去与未来，

让 Node.js 可以在 Serverless，低代码，SSR 等场景释放更大的价值。


五位重量级的嘉宾，

他们都是有代表作的开源大牛，

和 Node.js 新基建的贡献者，

这一次我们要挖掘低调实力派的无限潜力。


欢迎你和我们一起，畅想 2021。

本期嘉宾

hyj1991 @前端架构师

基于 Addon 的 Node.js 性能监控与线上故障定位解决方案

Easy-Monitor 3.0 版本开源了，旨在 Node.js 的开源生态工具链上，做一些能帮助到想使用和正在使用 Node.js 的开发者更好地感知自己的 Node.js 应用状态，以更好地面对来自性能和稳定性方面的挑战。

繁 易 @淘宝

Midway Hooks — 更具生产力的全栈框架

Midway Hooks 是社区Web方向比较有新意的项目，面向FaaS让开发更简单，面向React开发者，让同构全栈更简单。本次分享则将为大家介绍 Midway Hooks 的用法、设计理念以及背后的技术原理。

牧 曈 @阿里云

AirpackCDN — 新一代组件分发网络构建之路

Deno 带动了 http import 的发展，同时也使得社区从 Commonjs 到 ESM 的趋势日渐明显，这就促使社区会出现新一代 CDN 技术的诞生。AirpackCDN 是适用于现代 Web 应用程序的组件分发网络 (Components Delivery Network for modern web apps)，比如 Webpack5 快 4 倍。

冷 卉 @淘宝

iMove —开启可视化编排的新型开发模式

前端智能化解决了设计稿到代码的 UI 渲染部分，但逻辑代码如何生成依然是一个难题。iMove 是为开发者打造的支持可视化编排的低代码js框架，本次分享则将为大家介绍 iMove 的使用方法、底层原理、设计思想，以及此框架是如何解决开发中的痛点问题的。

张宇昂 @微信

SSR framework 5.0 - 同时支持 React, Vue 的服务端渲染能力

从 egg-react-ssr 到 ykfe/ssr 是一个不错的演进，Serverless-side render 概念也慢慢被大家接受，2020 年淘宝双十一使用了 ssr，拼多多的页面渲染也使用了 ssr，可见 ssr 是一个不可缺少的Web应用技术。SSR framework 2.0 以插件化的形式支持任意服务端框架与前端框架的结合，同时支持 Serverless 方式或传统 Node.js 应用方式部署。
